2016년 9~12월 삼성 아틱 경진 대회에 참가하였습니다.
사물 인터넷을 주제로 하여 우수상을 수상하였습니다.
개발된 플랫폼은 추가 배터리 없이 전선에 흐르는 전류에 의해 장비가 소모하는 전력을 측정해주는 기기 및 모니터링 시스템 입니다.

- 사용 소모 전력을 어떤 작은 기기가 할 수 있는데 추가 배터리 없이 전선에 부착만 해주면 에너지 하베스팅에 의해 작동이 됩니다.
- 해당 기기를 통해 장비의 전력이 데이터 통신 및 삼성이 개발한 아틱 장치를 통해 실시간으로 중계 됩니다.
- 에너지 소모 값이 크거나 이상 기후가 포착되면 릴레이를 통해 자동으로 장비를 끌 수 있습니다.
- 트위터를 통해 장비와 소통을 할 수 있고 에너지 사용량도 알 수 있고 트위터로 원력으로 제어를 할 수도 있습니다.
- 트위터 연동/ 앱 연동/ 노드레드(사물인터넷 최적화된 GUI 툴) 은 제가 담당하여 개발 하였습니다.
